# mermaid-mindmap

Slash command that turns a topic or outline into a Mermaid mindmap code block.

## What it provides

- **`/mindmap <topic>`** — generates a Mermaid `mindmap` block the chat
  renders inline. Pure prompt template; no network, no extra tools required.

## Example

```
/mindmap machine learning
```

Returns a fenced `mermaid` block with branches like "supervised learning",
"unsupervised learning", "reinforcement learning", etc.
